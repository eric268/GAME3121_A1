\hypertarget{index_Pong}{}\doxysection{Game made for GAME3121 at George Brown College F2021}\label{index_Pong}
This game demonstrates the use of object-\/oriented programming in C++ programming language through the development of the game Pong. It was created for the Game Engine Development class at George Brown College GAME3121 for the fall 2021 semester.

\begin{DoxyAuthor}{Author}
Eric Galway
\end{DoxyAuthor}
\begin{DoxyNote}{Note}
This game is made with Ogre Game Engine 
\end{DoxyNote}
\begin{DoxyAttention}{Attention}
The controls for the game are WASD for movement and R -\/key to restart the game 
\end{DoxyAttention}
